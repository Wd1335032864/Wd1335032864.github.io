网页基础概念和html认知

发表于 2022-04-29 分类于 前端学习
本文字数： 1.8k 阅读时长 ≈ 2 分钟
如题

我在干什么

记录一下黑马前端学习的过程

目标

认识网页组成和五大浏览器 明确web标准的构成，使用HTML骨架搭建出一个网页

网页

网页组成：图片，文字，视频。音频，超链接

网页本质：通过浏览器对代码的解析和渲染展示给用户的界面

浏览器

是网页显示、运行的平台，前端开发的利器
五大常见：IE 火狐 谷歌（这个好） Safari（mac自带） Opera（不太清楚）

渲染引擎

也叫浏览器内核
浏览器中专门对代码进行解析渲染的部分
不同浏览器内核是不同的
不同内核对相同代码的解析速度，性能，效果不一样
所以相同网页在不用浏览器中的显示效果不一定一样

web标准

目的：为了解决相同网页在不同浏览器下打开的效果不一样（浏览器内核不同）而制定的标准
三个构成： 结构：HTML 负责页面元素和内容（文字，图片等）
表现：CSS 网页元素的外观和位置等页面样式（颜色，大小等）
行为：JavaScript 网页模型定义与页面交互（与用户的交互行为，让网页能动起来）
eg： HTML 决定了人的骨架 CSS决定了人的样貌 JavaScript决定了人的动作

HTML概念

HTML（Hyper Text Markup Language） 超文本标记语言
超文本是一种组织信息的方式，它通过超级链接方法将文本中的文字、图表与其他信息媒体相关联

所以加粗要用到html 这里就是用到标记（标签）语法

<strong>表示后面这段文字要加粗</strong>表示到哪里结束 注意下划线 （有的标签有开始有结束，有的没有结束）

HTML 页面固定结构

网页类似与文章有它的基本架构
网页中的固定结构是要通过HTML标签描述的
即HTML骨架结构

整个网页通过 <html> </html> 包围住 title通过head标签包围 内容通过body标签包围（注意有开始有结束和下划线）

head为网页的头部 ，包括网页的标题（可能还有其他的~）
head里面 的title就是每个网页顶部的名字 如：

body里面就是网页的主体内容
